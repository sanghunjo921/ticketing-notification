## 프로젝트 개요
이 프로젝트는 티케팅 서비스에서 구매가 완료 된 시점에 rabbitMQ에 유저와 티켓 그리고 가격에 대한 정보가 담긴 메시지를 수신하여 해당 유저에게 이메일 발송을 목표로 합니다. 

- NestJS
- Postgresql
- TypeORM

## 사용 기술 스택
- 프레임워크: NestJS
- 데이터베이스: PostgreSQL
